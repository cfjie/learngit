一，初始化一个git本地库
命令：git init

二，添加文件到git本地库
命令：git add <filename>
命令：git commit -m <message>

三，查看本地库状态
命令：git status

四，查看修改内容
命令：git diff <filename>

五，查看版本历史
命令：git log

六, 查看命令历史
命令：git reflog

七，退回到历史版本
命令：git reset --hard <commit_id>

八，工作区的内容退回
命令：git checkout -- <filename>

九，暂存区的内容退回
命令：git reset HEAD <filename>

十，工作区内容还原到跟版本库一样
命令：git checkout -- <filename>

十一，删除文件
命令：git rm <filename>
命令：git commit m <message>

十二，创建SSHKEY
命令：ssh-keygen -t rsa -C "youremail@example.com"
登陆github，设置sshkeys


十三，将本地仓库同步到远程
命令：git remote add origin git@github.com:cjie/learngit.git
